🎮 JS GB Emulator

A from-scratch, single-file DMG (original GB) emulator written in plain JavaScript — built as an educational reference for how the LR35902 CPU, PPU, and memory map work together. Load your own legally-owned ROM file to play.

Scope note: this implements the full DMG CPU instruction set, background/window/sprite rendering, timers, joypad input, sound (all 4 APU channels via Web Audio), and ROM-only / MBC1 / basic MBC3 cartridges — enough to run many real games. It intentionally leaves out GB Color features and cycle-exact PPU/timer/APU edge cases to keep the source readable as a learning reference. Read the heavily commented source (view page source) to see how each piece works.
